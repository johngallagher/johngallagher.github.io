I'm sick of writing Javascript form validations.

I'm also sick of having to duplicate client side and server side code just to validate some freaking fields on a form.

If you are too, check out Parsley.

The most exciting thing for me here is the remote feature which allows a developer to hold all validation code server side.

Parsley also has, rather helpfully, validations for most common fields.

Finally - one place for validation rules! Most excellent.
